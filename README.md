# Clinton ThankGod — Graphics Design Portfolio

A personal portfolio website for a graphic designer, built to showcase brand identity, print, digital, and illustration work. Features a bold dark-mode aesthetic with vibrant accent colors.

## Tech Stack

- **Framework**: TanStack Start (React + Vite)
- **Routing**: TanStack Router (file-based)
- **Styling**: Tailwind CSS v4
- **Fonts**: Syne (display/headings) + Inter (body) via Google Fonts
- **Forms**: Netlify Forms
- **Hosting**: Netlify

## Key Pages

| Route | Description |
|---|---|
| `/` | Hero, featured projects, skills, CTA |
| `/projects` | Full portfolio gallery with category filtering |
| `/resume` | About page — bio, services, tools, experience |
| `/contact` | Project inquiry form |

## Running Locally

```bash
npm install
npm run dev
```

Runs on `http://localhost:3000`. For Netlify features (forms), run:

```bash
netlify dev
```
